## 1. 시작하기 전에

### 시스템 요구사항

| 항목 | 최소 사양 |
|------|-----------|
| 브라우저 | Chrome 90+ / Firefox 88+ / Edge 90+ |
| 화면 해상도 | 1280 × 720 이상 권장 |
| JavaScript | 활성화 필수 |

### 로컬 개발 환경 설치

```bash
# 1. 저장소 클론
git clone https://github.com/your-username/umamusmaeORDER.git

# 2. 프로젝트 디렉토리 이동
cd umamusmaeORDER

# 3. 의존성 설치
npm install

# 4. 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:5173` 에 접속하면 앱이 열립니다.

## 2. 앱 실행하기

앱을 처음 열면 **팀 등록 화면(TeamSetup)** 이 표시됩니다.

```
┌─────────────────────────────────┐
│         🏇 RaceOrder            │
│                                 │
│  팀을 등록하고 레이스를 시작하세요  │
│                                 │
│  [ + 팀 추가 ]                  │
│                                 │
│  [ 레이스 시작 ]  (2팀 이상 필요) │
└─────────────────────────────────┘
```

## 3. 팀 등록하기

### 3-1. 팀 추가

`+ 팀 추가` 버튼을 클릭하면 팀 입력 폼이 열립니다.  
**최소 2팀, 최대 12팀**까지 등록할 수 있습니다.

### 3-2. 입력 항목

| 항목 | 설명 | 예시 |
|------|------|------|
| **팀 이름** | 레이스에 표시될 이름 (필수) | `번개팀`, `폭풍기수` |
| **대표 색상** | 트랙에서 이 팀을 구분하는 색상 | 컬러 피커로 선택 |
| **아바타 스타일** | 트랙 위에 표시될 아이콘 모양 | ⚡ / 🌪 / 🔥 등 |
| **기초 스탯** | 속도 / 가속 / 스태미나를 슬라이더로 조정 | 아래 설명 참고 |
| **스킬 선택** | 보유할 스킬 1~2개 선택 | 아래 설명 참고 |

> 💡 **팁**: 스탯 합산이 제한되어 있어 모든 수치를 최대로 올릴 수 없습니다. 전략적으로 분배하세요.

### 3-3. 팀 수정 및 삭제

- 이미 추가된 팀 카드를 클릭하면 수정 모드로 진입합니다.
- 팀 카드 우측 상단의 `✕` 버튼으로 삭제할 수 있습니다.

## 4. 스탯 이해하기

각 팀은 3가지 기초 스탯을 가집니다.

### 속도 (Speed)

- 기본 이동 속도를 결정합니다.
- 수치가 높을수록 안정적으로 빠른 페이스를 유지합니다.
- **초반부터 선두를 달리고 싶다면** 속도를 높게 설정하세요.

### 가속 (Acceleration)

- 순간적인 속도 증가폭을 결정합니다.
- 스킬 발동 시 효과가 크게 나타납니다.
- **역전과 추월을 노린다면** 가속을 높게 설정하세요.

### 스태미나 (Stamina)

- 후반부 속도 유지력을 결정합니다.
- 스태미나가 낮으면 후반에 급격히 느려집니다.
- **막판 역전을 막고 싶다면** 스태미나를 높게 설정하세요.

### 컨디션 (Condition)

- 자동으로 랜덤 부여됩니다 (레이스 시작 시 결정).
- 최상 / 양호 / 보통 / 저조 / 최악 5단계로 구분됩니다.
- 컨디션이 낮으면 모든 스탯에 페널티가 적용됩니다.

## 5. 스킬 선택하기

각 팀은 레이스 중 발동할 스킬을 **최대 2개** 선택할 수 있습니다.

### 스킬 목록

| 스킬명 | 발동 조건 | 효과 | 지속 |
|--------|-----------|------|------|
| **선두 질주** | 1위 유지 중 | 속도 +10% | 5초 |
| **막판 스퍼트** | 잔여 거리 20% 이하 | 가속 대폭 상승 | 끝까지 |
| **추격의 불꽃** | 3위 이하 진입 시 | 순간 속도 부스트 | 3초 |
| **랜덤 부스트** | 무작위 타이밍 | 단기 속도 강화 | 2초 |
| **방해 저항** | 근접 경쟁 중 | 속도 감소 무효화 | 4초 |
| **침착한 페이스** | 전반전 | 스태미나 소모 감소 | 전반부 |

> ⚠️ **주의**: 같은 스킬을 여러 팀이 선택할 수 있습니다.

### 스킬 전략 예시

- **선두 지키기 전략**: 속도↑ + `선두 질주` + `방해 저항`
- **막판 역전 전략**: 스태미나↑ + `침착한 페이스` + `막판 스퍼트`
- **중반 추월 전략**: 가속↑ + `추격의 불꽃` + `랜덤 부스트`

## 6. 레이스 시작 및 진행

### 6-1. 레이스 시작

팀을 2팀 이상 등록한 뒤 `레이스 시작` 버튼을 클릭합니다.  
레이스 시작 전 **카운트다운(3 → 2 → 1 → GO!)** 이 표시됩니다.

### 6-2. 레이스 화면 구성

```
┌──────────────────────────────────────────────────────┐
│  경과 시간: 00:12          현재 1위: 번개팀           │
├──────────────────────────────────────────────────────┤
│ 레인 1  [번개팀  ⚡] ════════════════▶  74%          │
│ 레인 2  [폭풍기수 🌪] ══════════════▶  71%          │
│ 레인 3  [붉은불꽃 🔥] ═════════════▶  68%           │
│   ...                                                │
├──────────────────────────────────────────────────────┤
│ [스킬 로그]                                          │
│  00:08  폭풍기수의 "추격의 불꽃" 발동!               │
│  00:11  번개팀의 "선두 질주" 발동!                   │
└──────────────────────────────────────────────────────┘
```

### 6-3. 레이스 중 이벤트

레이스 진행 중 다음과 같은 이벤트가 발생할 수 있습니다.

| 이벤트 | 표시 방식 |
|--------|-----------|
| 스킬 발동 | 해당 레인 위에 스킬명 배너 팝업 |
| 추월 | 순위 변경 시 텍스트 로그 출력 |
| 막판 스퍼트 구간 | 잔여 20% 지점부터 배경 연출 변경 |
| 결승선 통과 | 통과 순서대로 순위 확정 및 기록 |

### 6-4. 레이스 일시 정지

레이스 진행 중 `일시 정지` 버튼으로 언제든지 멈출 수 있습니다.  
다시 `재개` 버튼을 누르면 이어서 진행됩니다.

## 7. 결과 화면 보기

모든 팀이 결승선을 통과하면 **결과 화면(ResultBoard)** 으로 전환됩니다.

### 7-1. 최종 순위

```
🏆 최종 순위

  🥇 1위  번개팀     기록: 00:42.3
  🥈 2위  폭풍기수   기록: 00:43.1
  🥉 3위  붉은불꽃   기록: 00:44.8
     4위  ...
```

### 7-2. 레이스 통계

| 항목 | 내용 |
|------|------|
| **MVP** | 가장 많은 스킬을 발동한 팀 |
| **최다 추월** | 레이스 중 가장 많이 순위를 올린 팀 |
| **최고 순간 속도** | 특정 틱에서 가장 빠른 이동거리를 기록한 팀 |
| **컨디션 행운** | 가장 높은 컨디션을 받은 팀 |

### 7-3. 스킬 발동 로그 전체 보기

결과 화면 하단의 `전체 로그 보기` 버튼을 클릭하면 레이스 중 발생한 모든 스킬 이벤트를 시간순으로 확인할 수 있습니다.

## 8. 다시 달리기

결과 화면에서 두 가지 방식으로 다시 시작할 수 있습니다.

| 버튼 | 동작 |
|------|------|
| **같은 팀으로 다시 달리기** | 팀 구성을 유지하고 레이스만 새로 시작 (컨디션 재추첨) |
| **처음부터 시작** | 모든 팀 정보를 초기화하고 팀 등록 화면으로 돌아감 |

## 9. 자주 묻는 질문 (FAQ)

**Q. 팀을 12팀 이상 등록할 수 없나요?**  
A. 현재 최대 12팀으로 제한되어 있습니다. 화면 가독성과 레이스 연출 품질을 위한 설계 결정입니다.

**Q. 같은 팀 이름을 사용할 수 있나요?**  
A. 중복 이름은 허용되지 않습니다. 고유한 팀 이름을 입력해주세요.

**Q. 레이스 결과가 매번 달라지나요?**  
A. 네. 컨디션 추첨과 스킬 발동 타이밍에 랜덤 요소가 포함되어 있어 매 레이스마다 다른 결과가 나옵니다.

**Q. 스탯을 어떻게 배분하는 게 유리한가요?**  
A. 정답은 없습니다. 단, 스태미나를 너무 낮게 설정하면 초반 선두를 달리다가 후반에 급락하는 경우가 많습니다.

**Q. 모바일에서도 사용할 수 있나요?**  
A. 현재는 PC 화면(1280px 이상)에 최적화되어 있습니다. 모바일 대응은 추후 업데이트 예정입니다.
